## Why

Пользователям нужна страница авторизации для входа в приложение. Маршрут `/login` уже существует в роутере, но LoginPage — заглушка. Необходимо сверстать полноценную страницу входа по дизайну Figma.

## What Changes

- Подключить react-router по адресу `/login` (маршрут уже настроен)
- Создать страницу LoginPage с полной вёрсткой по макету
- Реализовать форму авторизации: поля логин и пароль, чекбокс «Запомнить данные», кнопка «Войти», ссылка «Создать» для регистрации
- Использовать существующие компоненты: Input, Checkbox, Button, Icon

## Capabilities

### New Capabilities

- `login-page`: страница авторизации на маршруте `/login` с формой входа (логин, пароль, «Запомнить данные», кнопка «Войти», разделитель «или», ссылка «Нет аккаунта? Создать»)

### Modified Capabilities

- `app-structure`: маршрут `/login` уже определён; LoginPage получает полноценную реализацию вместо заглушки

## Impact

- `src/pages/LoginPage/` — новая вёрстка и стили
- `src/app/router.tsx` — маршрут `/login` уже настроен, изменений не требуется
- Существующие компоненты Input, Checkbox, Button, Icon — используются без изменений
- CSS variables и CSS modules — для соответствия дизайну Figma
